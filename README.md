# kanjimag

## NAME
kanjimag - Kanji Magnifier

## DESCRIPTION
Show characters in an enlarged view.  
You can display complex characters?such as kanji with many strokes?largely on the screen to inspect them clearly.

I developed this app because I needed something similar to the KANJI CHECK feature that was included on KYOCERA's DIGNO mobile phones.
